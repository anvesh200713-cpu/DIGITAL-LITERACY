PROJECT OVERVIEW
This repository serves as a comprehensive digital portfolio created for the Student Digital Ambassador project. The primary objective is to demonstrate digital literacy, establish a professional online brand, and promote cybersecurity awareness among college peers. By completing these five modules, I am building a strong foundation for my academic journey and future career goals, including my plans to pursue an M.Tech through the GATE 2028 exam. This repository reflects my commitment to technical excellence, effective communication, and responsible digital citizenship.

REPOSITORY STRUCTURE & MODULE DETAILS

FOLDER 1: task-1-presentation/
- Contains: A visual infographic designed to educate first-year students on digital literacy.
- Description: This module focuses on visual communication. It translates complex concepts regarding digital safety, professional online presence, and productivity tools into an accessible, easy-to-understand visual format.

FOLDER 2: task-2-portfolio/
- Contains: Screenshots of newly established professional profiles on GitHub, LinkedIn, and Kaggle.
- Description: This folder showcases the foundation of my professional digital footprint. These platforms will serve as the central hub for my career progression, hosting my technical coursework, side projects (such as my recent Python Stone-Paper-Scissor game), and connecting me with the broader academic and AI/ML community.

FOLDER 3: task-3-platforms/
- Contains: Documentation of my Python coding practice on HackerRank and screenshots of a peer-focused Google Form Quiz alongside its linked Google Sheet.
- Description: This module highlights my active engagement with foundational programming logic and cloud collaboration tools. Mastering these platforms is essential for efficiently coordinating group projects, gathering data, and passing future technical assessments.

FOLDER 4: task-4-email-etiquette/
- Contains: Drafts of standard professional emails (a deadline extension request and an internship inquiry) and a "Social Media Do's and Don'ts" checklist.
- Description: Effective digital communication is just as critical as writing clean code. This folder demonstrates my ability to navigate professional academic correspondence, utilize correct email formatting, and maintain a respectful, career-oriented social media presence.

FOLDER 5: task-5-cybercrime/
- Contains: A detailed case study on UPI fraud targeting students and a comprehensive cybersecurity prevention checklist.
- Description: This module addresses the critical need for online safety. It provides a realistic breakdown of how modern phishing scams operate and offers actionable advice for protecting financial data, including the proper use of the National Cyber Crime Portal (Helpline 1930).

REPORT FOLDER: report/
- Contains: The final, comprehensive Project Report.
- Description: A full synthesis of my learnings, detailed reflections on the specific digital tools utilized, and a deep dive into how these newly formed habits will positively shape my academic trajectory.
